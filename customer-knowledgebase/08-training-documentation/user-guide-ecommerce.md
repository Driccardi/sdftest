## Getting Started

### Creating an Account

**Benefits of Creating an Account**:
- Faster checkout
- Order history and tracking
- Saved addresses and payment methods
- Wishlist and favorites
- SunStyle Rewards loyalty program
- Exclusive offers and early access

**How to Create an Account**:
1. Click "Sign Up" in the top right corner
2. Enter your email address
3. Create a password (minimum 12 characters)
4. Optional: Add your name, phone number
5. Click "Create Account"
6. Verify your email (check inbox for verification link)

**Account Security**:
- Use a strong, unique password
- Never share your password
- Enable two-factor authentication (recommended)
- Log out when using shared devices

### Guest Checkout

**Quick Checkout Without Account**:
- Shop and checkout without creating an account
- Provide email for order confirmation
- Track your order via link in confirmation email

**Note**: Some features require an account (wishlist, rewards, saved addresses)

## Returns and Exchanges

### Return Policy

**30-Day Return Policy**:
- Return within 30 days of delivery
- Items must be unworn with original tags
- Original packaging included
- Full refund or exchange

**Non-Returnable Items**:
- Prescription eyewear (unless defective)
- Items worn or damaged by customer
- Items without original packaging

### How to Return

**Online Returns**:
1. Log in to your account
2. Go to "My Orders"
3. Select order to return
4. Click "Return Items"
5. Select items and reason
6. Choose refund method (original payment, store credit, exchange)
7. Print prepaid return label
8. Pack items securely
9. Drop off at carrier location
10. Receive refund within 3-5 business days after receipt

**In-Store Returns**:
1. Bring item with receipt or order confirmation
2. Visit any SunStyle Retail store
3. Associate will process return
4. Immediate refund or exchange

**Return Shipping**:
- **Free return shipping** (prepaid label provided)
- Drop off at FedEx, UPS, or USPS location

### Exchanges

**Exchange Process**:
- Process as return
- Place new order for desired item
- Or contact customer service for assistance
