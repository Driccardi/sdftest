# 010.020 - Subsidiary Structure Configuration

## Configuration Metadata
- **Phase**: 010 - Record-to-Report
- **Component**: Subsidiary Structure (OneWorld)
- **Priority**: Critical
- **Estimated Effort**: 6 hours
- **Dependencies**: None (foundational configuration)

## Business Context

### Purpose
Establish a multi-subsidiary structure to support SunStyle Retail's current USA operations and future Canada expansion. The subsidiary structure enables separate legal entity reporting, multi-currency operations, tax compliance by jurisdiction, and consolidated financial statements.

### Business Requirements
- **Current**: Single operating entity in USA ($45M revenue)
- **Future**: Canada expansion planned (2027)
- **Compliance**: Separate tax reporting by country
- **Reporting**: Consolidated and standalone financials
- **Currency**: USD (primary), CAD (future)

### Key Stakeholders
- **Patricia Wong**, CFO (Approval)
- **Michael Thompson**, CEO (Strategic approval)
- **Susan Park**, Controller (Implementation)
- **External Auditors** (Compliance review)

## Implementation Steps

### Phase 1: Parent and USA Subsidiary (Week 1)

**Step 1: Enable OneWorld Features** (1 hour)
1. Verify OneWorld is enabled (already active per CKB)
2. Enable multi-subsidiary features
3. Set global preferences

**Step 2: Create Parent Company** (1 hour)
1. Create SunStyle Retail Corporation subsidiary record
2. Enter legal and address information
3. Configure as parent company
4. Set fiscal calendars

**Step 3: Create USA Operating Subsidiary** (2 hours)
1. Create SunStyle Retail USA LLC subsidiary record
2. Link to parent company
3. Configure financial settings
4. Set base currency (USD)
5. Configure default accounts
6. Enable features (inventory, billing, shipping)

**Step 4: Assign Chart of Accounts** (1 hour)
1. Assign all accounts to USA subsidiary
2. Configure parent-level consolidation accounts
3. Verify account accessibility

**Step 5: Configure Locations** (1 hour)
1. Map warehouse to USA subsidiary
2. Map corporate HQ to USA subsidiary
3. Map all 25 stores to USA subsidiary
4. Verify location-subsidiary relationships

### Phase 2: Future Canada Subsidiary (2027)

**Prerequisites**:

## Business Rules

### Subsidiary Assignment Rules
1. All transactions default to USA subsidiary
2. Location determines subsidiary (when Canada active)
3. Customer/vendor assigned to primary subsidiary
4. Cross-subsidiary transactions require approval

### Consolidation Rules
1. Monthly consolidation after subsidiary closes
2. Intercompany eliminations automated
3. Currency translation at period-end rates
4. Consolidation adjustments documented

### Reporting Rules
1. Subsidiary reports due 3 days after period close
2. Consolidated reports due 5 days after period close
3. Variance analysis required for >5% changes
4. Board reporting includes consolidated and USA detail
