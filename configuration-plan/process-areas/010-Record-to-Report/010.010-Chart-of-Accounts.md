## NetSuite Configuration

### Account Structure

#### Account Numbering Schema
Format: `X-XXXX-XXX-XXX`
- Position 1: Account Type (1 = Asset, 2 = Liability, 3 = Equity, 4 = Income, 5 = COGS, 6 = Expense, 7 = Other Income, 8 = Other Expense)
- Positions 2-5: Major Category
- Positions 6-8: Subcategory
- Positions 9-11: Detail (optional)

Example: `4-1000-100-001` = Revenue - Product Sales - Premium Sunglasses - Ray-Ban

#### Account Configuration Template

**Navigation**: Setup > Accounting > Accounts > New

**Standard Fields**:
- Account Name: Descriptive name
- Account Number: Per numbering schema
- Account Type: NetSuite standard types
- Subsidiary: USA (primary), add Canada when ready
- Currency: USD (USA), CAD (Canada future)
- Include Children: Checked for parent accounts
- Summary: Checked for rollup accounts
- Inactive: Unchecked (active)

## Business Rules

### Account Usage Rules
1. All revenue must be posted to 4-XXXX accounts
2. COGS must match revenue structure (same subcategories)
3. Expenses require Department and Location coding
4. Store-specific costs use Class dimension
5. Intercompany accounts for future Canada operations

### Posting Rules
1. No manual journal entries without approval
2. Monthly close process locks prior periods
3. Depreciation automated monthly
4. Accruals posted last day of month
5. Reclassifications require FP&A approval

### Security and Controls
1. Account creation: Controller only
2. Account modification: CFO approval
3. Account inactivation: Documented reason required
4. Audit trail enabled for all changes
