IMPLEMENTATION STEPS
====================

PHASE 1: NetSuite Setup (Week 1-2)
-----------------------------------
1. Upload brand assets to File Cabinet
2. Create email template records (Customization > Forms > Email Templates)
3. Replace all [PLACEHOLDER] values with actual URLs
4. Test Freemarker variables with sample data
5. Create plain text versions
6. Configure email preferences (Setup > Company > Email Preferences)
7. Set up SPF, DKIM, DMARC authentication

PHASE 2: Workflow Configuration (Week 2)
-----------------------------------------
Create workflows for each transactional email:
- Order Confirmation: Sales Order -> Pending Fulfillment
- Shipment Notification: Item Fulfillment -> Shipped
- Delivery Confirmation: Daily scheduled (requires carrier webhook)
- Return Label: Return Authorization -> Approved
- Refund Processed: Customer Refund -> Complete
- Loyalty Balance: Monthly scheduled (1st of month)
- Back-in-Stock: Hourly scheduled inventory check

PHASE 3: Klaviyo Integration (Week 2-3)
----------------------------------------
1. Connect Klaviyo to NetSuite (via Celigo or API)
2. Sync customer data, products, orders
3. Create Abandoned Cart flow (3 emails: 24h, 48h, 5 days)
4. Create Birthday Email flow (7 days before birthday)
5. Test both flows with sample data

PHASE 4: Testing (Week 3-4)
----------------------------
Test each template in:
- Gmail (web, mobile)
- Outlook (desktop, web, mobile)
- Apple Mail (Mac, iPhone)
- Yahoo Mail
Check: responsiveness, links, variables, images, spam score

PHASE 5: Launch & Monitor (Week 4+)
------------------------------------
- Soft launch with 10% of customers
- Monitor metrics daily
- Ramp to 100% over 1 week
- Weekly optimization reviews

KEY METRICS
===========
- Open Rate: >40% (transactional), >25% (marketing)
- Click Rate: >10% (transactional), >3.5% (marketing)
- Bounce Rate: <2%
- Spam Rate: <0.1%

FREEMARKER VARIABLES
====================
Customer: firstname, lastname, email, custentity_loyalty_tier, custentity_loyalty_points
Transaction: tranid, trandate, total, subtotal, taxtotal, shippingcost, shipmethod
Custom: custbody_fulfillment_method, custbody_loyalty_points_earned, custbody_tracking_number
Items: Loop through transaction.item for product details
